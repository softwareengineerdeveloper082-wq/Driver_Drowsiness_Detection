# Real-Time Driver Drowsiness Detection System 🚗💤

An intelligent, computer-vision-based safety application designed to prevent road accidents caused by driver fatigue and micro-sleep.

## 🌟 Key Features
- **Real-Time Eye Tracking:** Maps and analyzes facial geometry smoothly using a live camera stream.
- **Biometric Evaluation:** Computes the Eye Aspect Ratio (EAR) continuously to monitor blink rates and prolonged closure.
- **Proactive Audio Warning:** Triggers an immediate audible alert to snap the driver awake if their eyes remain closed beyond a safe threshold.
- **Lightweight Design:** Built without massive external audio overhead by utilizing native system sounds.

## 🛠️ Built With
* [Python](https://www.python.org/) - Programming language
* [OpenCV](https://opencv.org/) - Real-time video processing
* [MediaPipe](https://google.github.io/mediapipe/) - Advanced facial landmark mesh tracking
